This project analyzes Udemy course data to uncover key trends and insights related to pricing, ratings, enrollments, and course levels. The goal is to understand what factors contribute to higher engagement and success on the Udemy platform, and to visualize these findings through exploratory data analysis (EDA).

---

## Problem Statement
Online learning platforms like Udemy host thousands of courses, but not all perform equally in terms of enrollments, ratings, and popularity.  
This project aims to answer the following questions:
1. What factors influence course popularity and enrollment?
2. Are there significant differences between free and paid courses?
3. How do course levels (Beginner, Intermediate, Expert) impact engagement?
4. What trends can be observed in course publishing dates?
5. Are there outliers or patterns in pricing, ratings, or enrollments?
